# 2Match | Фронтенд

## Требования перед запуском

[Node.js](https://nodejs.org/en/download) ≥ v22.17.0

## Запустить локально

1. Установите зависимости

- npm

  ```bash
  npm i
  ```

- yarn

  ```bash
  yarn install
  ```

2. Запустите приложение в dev-режиме

- npm

  ```bash
  npm run dev
  ```

- yarn

  ```bash
  yarn dev
  ```
